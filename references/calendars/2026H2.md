# 2026H2 周次日历

## 规则

- `H2` 表示下半年。
- `W` 表示 Week。
- 任务周按周一到周日计算。
- 核心基准规则：当前这周就是 `2026H2W1`，也就是 `2026-07-06` 至 `2026-07-12`；后续每过一个任务周，周次在这个基准上递增。
- 不按自然年周数重新计算，也不把 2026 年 1 月的第一周当作 `2026H2W1`。

## 回答规则

当用户问“本周打卡任务是什么”时：

1. 先用当前日期匹配上面的日期范围。
2. 回答匹配到的周次和日期范围。
3. 如果 `references/weeks/<周次>.md` 存在，读取并输出该周详细任务。
4. 如果周任务文件不存在，从 `references/plans/2026H2.md` 的周任务总表中输出该周标题、目标和主要产出。
